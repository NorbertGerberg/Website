41
42 ext : 0 - regular assembler instruction mnemonic

44 2 - not an assembly instruction
45
46 class : 0 - special BRK,JAM,NOP : nothing affected

48 2 - affects A only
49 3 - affects X only
50 4 - affects Y only
51 5 - affects memory
52
53 9 - not dealt with
54*/
