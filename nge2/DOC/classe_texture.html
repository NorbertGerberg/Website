Detailed Description

Texture flags: xyzw x: 0 -> Mirror 1 -> Border 2 -> Clamp y: 0 -> border color shift 1 -> border color mask z: 0 -> compare greater 1 -> compare not equal 2 -> compare never 3 -> compare always 4 -> compare shift w: 0 -> min/mag point 1 -> min/mag mask 2 -> min/mag shift 3 -> min/mag anisotropic Texture class
