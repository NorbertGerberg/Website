About

NGFramework is a C# game framework designed for easy 2D application development.
The Runtime Engine is written in C++ and uses Mono C# for scripting!

With ngf, you can easily create a window, draw sprites, receive user input, and play audio files!
It is completely free to use; you do not have to pay any royalties!

Download

After installing the Runtime Engine, execute "create association.bat" as an administrator to establish associations
for all files with the ".ngf" extension to the runtime application.
